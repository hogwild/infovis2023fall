Welcome to our interactive exploration of the human brain's connectivity network!

This project represents a simplified visualization of the complex connectivity in a human brain. Note that connectivities are in vector forms. By directly hovering the mouse on the main 3-d brain, you can see how one node projects information to and receives information from other nodes. By utilizing the subplots, you are also allowed to explore how each node's source property is differ from target property, as well as the connection strength between certain nodes.
